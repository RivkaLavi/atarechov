Installation

Uploading Javo via FTP to Wordpress

  • Download the zipped theme pack to your local computer from themeforest and extract the ZIP file contents to a folder on your local computer.
  • In the extracted folder you will find 'listopia.zip' Extract this file .
  • Using an FTP client to access your host web server
  • Upload the unzipped theme folder in wp-content / themes.
  • Visit Administration > Appearance > Themes, under the Manage Themes tab, under the Available Themes, if necessary navigate to the page displaying that Theme, then click the Activate link to make the Theme, the Current Theme.
More information about installation themes

Installing Javo From the WordPress Dashboard

  • Download the zipped theme pack to your local computer from themeforest and extract the ZIP file contents to a folder on your local computer.
  • Log into your WordPress admin panel and click on the “Appearance” tab.
  • Now click on the "Add New Themes" tab.
  • Click on the "Upload" button, "Browse" for file called listopia.zip In the extracted folder select "Open".
  • Click the "Install Now" button.
  • Activate the theme from this page, OR "Return to Themes Page" and activate from there

Theme Setup Wizard

  • When you activate our theme, you will see the Setup Wizard screen.
  • This will help you with the process you need to use our theme.
  • Please click the "Let’s go!" button.

Installing Plugins

  • You can install the necessary plugins for our theme here.
  • Please click the "Continue" button.
  • Plugins are automatically installed and activated.
  • You will be moved to the next process after this operation is completed.

Status

  • You should set the PHP version, Post Max Size, and WP MEMORY LIMIT as we recommend.
  • If these are low, you may have problems importing demo data or running the site continuously.
  • WP debug is an optional option. You can go to the next page by clicking the "Next" button.

Installing Sample Content

  • Please choose the demo version and import type you want.
  • We recommend that you select "All" as the import type. So you can import all our data from the demo version you have selected.
  • And you should check "Do you want to import media files?" Then click the "Import" button.

  • Once the import starts, the gauge below will be filled. If the import is completed normally, it will become 100%.
  • There might be some time this gauge stops with a certain numeral.
  • This happens because the upload size which is set up in your site is low.
  • You should increase the upload size after consulting your server on this issue. Detail

Setup Front Page

  • Please move to Dashboard >> Settings >> Reading.
  • Please select 'A static page (select below)' and select the front page you want from the dropdown.

Setup Permalink

  • Please move to Dashboard >> Settings >> Permalinks.
  • Please choose ‘Post name’ and save it.

Setup Menu

  • Please move to Dashboard >> Settings >> BuddyPress.
  • You have to check the elements tab except for 'Site Tracking' in the Components tab. So that you can put more buddypress pages in the menu.

  • Please move to Dashboard >> Appearance >> Menus.
  • If there is not a menu, please input Menu name, click the button "Create Menu" on the right side, and make it.
  • Please check a select box in the Menu settings according to the menu's purpose.
  • Please choose on the left the items you want to add to the menu and click the button 'Add to Menu'.
  • Once you click the button 'Save Menu' and save it, it will appear in your site.

  • You can see various ‘Display location’ under Menu Settings. Our theme can add menus in several places.

My menu nav

  • You can add the 'My menu' in the 'Javo' tab to the menu. This is the menu associated with the currently logged in users.
  • They can navigate to the edit profile or logout through this menu.
  • And they can navigate to the menus that admin has set with My menu nav.

  • If you check the ‘My menu nav’ in the Display location and create a menu, it will be added under My menu.

  • Basically, you should add BuddyPress, BBPress related menus here. Then users can access easily the pages associated with them.

Top Nav Menu - Left

  • Please go to Dashboard >> Appearance >> Menus. And create a new menu and check ‘Top Nav Menu – Left’ in ‘Display location’.

  • And please add the menus you want here.

  • Then, you can see the menus you have added on the left side of the header.

Top Nav Menu - Right

  • Please go to Dashboard >> Appearance >> Menus. And create a new menu and check ‘Top Nav Menu – Right’ in ‘Display location’.

  • And please add the menus you want here. Our demo site is using our Javo menus here.

  • Then, you can see the menus you have added on the right side of the header.

Setup Widget

  • Please move to Dashboard >> Appearance >> Widgets.
  • The left ones are the widgets which you can use in our theme, and the right ones widget area.
  • Please put the widget you want in the place you want, and set up the widget options.
